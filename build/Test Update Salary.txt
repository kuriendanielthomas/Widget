Upd_Salary is a database procedure to either increase or decrease salary of a particulare employee.
This procedure requires following  parameters.
1.Parameter Employee ID.
2.Percentage of salary update as integer.
3.Type of update . Pass 'I' for Increment and 'D' for Decrement.
4.Out parameter for for catching error messages.

Testcase 1
Passing invalid parameter type for update. This code should generate an error message for wrong type.
declare
ErrorMessage  varchar2(120);
begin
UPD_SALARY(90001,10,'L',error_message);
dbms_output.put_line(ErrorMessage);
end;

Testcase 2
Passing either 0 or negetive value of percentage for update. This code should generate an error message.
declare
ErrorMessage  varchar2(120);
begin
UPD_SALARY(90001,0,'I',error_message);
dbms_output.put_line(ErrorMessage);
end;


Testcase 3
Passing invalid parameter for employee ID. This code should generate an error message for wrong employee ID.
declare
ErrorMessage varchar2(120);
begin
UPD_SALARY(90052,10,'I',error_message);
dbms_output.put_line(ErrorMessage);
end;
